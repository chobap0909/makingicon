무료 아이콘 변환기

이미지를 원하는 영역으로 자른 뒤 사각형 또는 원형 아이콘으로 변환하세요. 원하는 크기만 선택해 ICO, ICNS, PNG 파일로 다운로드할 수 있습니다.

1 이미지 선택

이미지를 끌어오거나 클릭해서 업로드

PNG, JPG, WebP, GIF, SVG, BMP 지원 · 최대 20MB 권장
2 영역과 모양 선택
모양
이미지 위에서 드래그해 아이콘으로 만들 영역을 선택하세요.
3 크기 선택

미리보기를 누르면 해당 크기 하나가 선택됩니다. 선택한 크기는 ICO와 PNG 다운로드에 적용됩니다.

4 다운로드
주요 기능

크기 직접 선택

16, 32, 48, 64, 128, 256px 미리보기를 눌러 필요한 크기만 포함할 수 있습니다.

사각형·원형 모드

일반 앱 아이콘부터 프로필용 원형 아이콘까지 투명 배경으로 만들 수 있습니다.

브라우저 처리

이미지를 서버에 올리지 않고 사용자의 기기에서 바로 변환합니다.

ICO / ICNS / PNG

Windows, macOS, 웹사이트 파비콘과 일반 이미지 아이콘 작업에 활용할 수 있습니다.

사용법
  1. 이미지를 업로드합니다.
  2. 사각형 또는 원형 모드를 선택합니다.
  3. 이미지 위에서 아이콘으로 만들 영역을 드래그합니다.
  4. 미리보기에서 필요한 크기를 선택합니다.
  5. ICO, ICNS 또는 PNG 파일로 다운로드합니다.
자주 묻는 질문
ICO와 ICNS의 차이는 무엇인가요?
ICO는 Windows와 웹 파비콘에서 많이 쓰이고, ICNS는 macOS 앱과 Finder에서 사용하는 아이콘 형식입니다.
원형 아이콘을 만들 수 있나요?
네. 원형 모드를 선택하면 선택 영역이 원형으로 마스킹되고 바깥 영역은 투명 배경으로 저장됩니다.
이미지가 서버로 업로드되나요?
아니요. FileReader와 Canvas API를 사용해 브라우저 안에서 이미지를 처리합니다.
파비콘으로 바로 사용할 수 있나요?
다운로드한 ICO 파일명을 favicon.ico로 바꾸고 사이트 루트 폴더에 올리면 일반적인 브라우저 파비콘으로 사용할 수 있습니다.
